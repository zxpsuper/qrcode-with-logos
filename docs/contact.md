# Contact Us

We'd love to hear from you! Whether you have questions, feedback, suggestions, or just want to say hello, there are several ways to get in touch with us.

## GitHub

### Issues
For bug reports, feature requests, or technical questions, please open an issue on our GitHub repository:

[Open an Issue](https://github.com/zxpsuper/qrcode-with-logos/issues/new)

### Discussions
Join our GitHub Discussions to connect with other users, share ideas, ask questions, or showcase how you're using qrcode-with-logos:

[Join Discussions](/guide/discussions)

### Pull Requests
If you'd like to contribute code directly, feel free to submit a pull request:

[Submit a Pull Request](https://github.com/zxpsuper/qrcode-with-logos/pulls)

## How to Get Help

1. **Check the Documentation**: First, make sure you've read our [Getting Started](/guide/getting-started) guide and [Options](/guide/options) documentation.
2. **Search Issues**: Someone might have already reported the same issue or asked the same question.
3. **Open a New Issue**: If you can't find a solution, open a new issue with a clear description and, if possible, a reproduction example.

## Business Inquiries

For business-related inquiries, partnerships, or sponsorship opportunities, please contact us via email with "Business Inquiry" in the subject line.

## Response Time

We try our best to respond to all inquiries in a timely manner. For GitHub issues, you can typically expect a response within a few days.
