# About qrcode-with-logos

## What is qrcode-with-logos?

qrcode-with-logos is a powerful and easy-to-use JavaScript library that allows you to generate beautiful QR codes with custom logos and extensive styling options. Built on top of the popular `qrcode` library, it extends functionality with additional customization features that make your QR codes stand out.

## Our Mission

We aim to provide developers with a simple yet flexible tool to create visually appealing QR codes that can incorporate brand identities through logos, custom colors, and various styling options. Whether you're building a website, mobile app, or marketing campaign, qrcode-with-logos helps you create QR codes that match your design aesthetic.

## History

qrcode-with-logos was created in 2019 by [zxpsuper](https://github.com/zxpsuper) as a side project to solve the need for customizable QR codes with logos. Since then, it has grown into a widely used library with thousands of downloads per month on npm.

## Features

- **Logo Integration**: Easily add logos to your QR codes
- **Customizable Dots**: Multiple dot styles and color options
- **Corner Styling**: Different corner types and colors
- **Color Customization**: Full control over colors
- **Download Support**: Built-in methods to download QR codes
- **Browser-based**: Uses Canvas API for high-quality rendering

## Tech Stack

- **JavaScript**: Core language
- **Canvas API**: For QR code rendering
- **qrcode**: Base dependency for QR code generation
